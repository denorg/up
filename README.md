# 🔌 Up

Check if a website is up right now.

```ts
import { isUp } from "https://raw.githubusercontent.com/denorg/up/master/mod.ts";

const result = await isUp("https://google.com"); // true
```

Required permissions:

1. `--allow-net`

## 👩‍💻 Development

Run tests:

```bash
deno test --allow-net
```
